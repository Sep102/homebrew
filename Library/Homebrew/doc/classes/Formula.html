patches()

patches are automatically applied after extracting the tarball return an array of strings, or if you need a patch level other than -p1 return a Hash eg. 

  {
    :p0 => ['http://foo.com/patch1', 'http://foo.com/patch2'],
    :p1 =>  'http://bar.com/patch2',
    :p2 => ['http://moo.com/patch5', 'http://moo.com/patch6']
  }

The final option is to return DATA, then put a diff after END. You can still return a Hash with DATA as the value for a patch level key.
